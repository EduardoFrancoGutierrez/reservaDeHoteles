Reserva de habitaciones

Elige las habitaciones más acordes con tus necesidades, a partir de un listado de hoteles disponibles. Nuestras habitaciones son individuales, dobles, triples o suites. En función del tipo de habitación, cada una dispone de diferentes servicios como Internet, TV, Jacuzzi o aire acondicionado.

Tras la elección, podrás reservar cada habitación durante un número de horas determinado, siempre que dicha habitación se encuentre disponible para reservas. Se acabó pagar las habitaciones para 24 horas como mínimo. Cada cliente y gestor de hotel, reciben una notificación vía mail de la reserva realizada.


Visualizar reservas

Desde el menú principal de la aplicación, dispones de opciones para visualizar las reservas de habitaciones realizadas en cada hotel. A partir de ahora, el gestor de cada hotel tendrá de primera mano el porcentaje de ocupación de cada hotel, de cada a su administración.

También, cada cliente que quiera reservar una habitación, tendrá la posibilidad de ver este listado y contrastar si se han realizado más reservas de habitaciones, pudiendo analizar la popularidad del mismo.

Cancelación de reservas

Desde el menú principal de la aplicaci&oaute;n, cada cliente dispone de una funcionalidad para cancelar una reserva realizada. Antes de realizar una cancelación, se aconseja revisar la política de cancelación de reservas de cada hotel, ya que cada uno decide la penalización que aplica a cada cliente, tras cancelación.

Al igual que el cliente, un gestor de hotel por causas de fuerza mayor, puede cancelar una reserva. En todos los casos de cancelación de reserva, el cliente y el gestor del hotel reciben una notificación de la cancelación. Gracias a esta funcionalidad, el cliente puede realizar rápidamente una reserva en otro hotel próximo.
